# Student Management System (Spring MVC)

A web-based Student Management System developed using Spring MVC.  
This application allows users to manage student records efficiently with CRUD operations and search functionality.

This project is created as an **academic / learning project** to understand Spring MVC architecture and layered application design.

## How to Run the Project

  1. Clone the repository:

      git clone https://github.com/your-username/studentapp.git


  2. Open the project in your IDE.

  3. Configure database details in application.properties.

  4. Build the project using Maven:

      mvn clean install

  5. Deploy the project on Apache Tomcat or run directly if configured via IDE.

  6. Open your browser and visit:

      http://localhost:8080/
